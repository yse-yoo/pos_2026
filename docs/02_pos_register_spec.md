# レジシステム メイン画面 仕様書

**バージョン:** 1.0.0  
**作成日:** 2026-04-17  
**対象ファイル:** `pos_register.html`

---

## 1. 概要

クレープ屋さんのPOS（Point of Sale）レジシステムのメイン画面。商品の選択・カート管理・会計処理を1画面で完結できるUIモックアップ。

---

## 2. 画面構成

画面は左右2カラムのレイアウトで構成される。

```
┌──────────────────────────────┬────────────────┐
│                              │                │
│  左パネル（商品選択エリア）     │ 右パネル        │
│  - ヘッダー／時計             │ （注文内容）     │
│  - カテゴリタブ               │                │
│  - 商品グリッド               │                │
│                              ├────────────────┤
│                              │ 合計サマリー    │
│                              ├────────────────┤
│                              │ 会計ボタン群    │
└──────────────────────────────┴────────────────┘
```

---

## 3. 左パネル仕様（商品選択エリア）

### 3.1 ヘッダー

| 要素 | 内容 |
|------|------|
| タイトル | 「レジシステム」固定テキスト |
| 時計 | 現在時刻をリアルタイム表示（HH:MM:SS、1秒間隔更新） |

### 3.2 カテゴリタブ

商品をカテゴリで絞り込むフィルタタブ。

| タブ名 | 対象商品 |
|--------|----------|
| 全て | すべての商品 |
| フード | おにぎり、サンドイッチ、弁当、パン |
| ドリンク | お茶、コーヒー、水、ジュース |
| トッピング | 雑誌、電池 |
| 有料袋 | ティッシュ、歯ブラシ |

- アクティブなタブは青色（`#1d4ed8`）でハイライト
- タブ切り替えで商品グリッドがリアルタイム更新

### 3.3 商品グリッド

- 3カラムのグリッドレイアウト
- 各商品カードに絵文字アイコン・商品名・単価を表示
- クリックでカートに追加（同一商品の場合は数量+1）

## 4. 右パネル仕様（注文内容）

### 4.1 ヘッダー

| 要素 | 内容 |
|------|------|
| タイトル | 「注文内容」固定テキスト |
| 伝票番号 | `伝票 #0001` 形式、会計完了ごとに自動採番（4桁ゼロ埋め） |

### 4.2 カートアイテム一覧

カートに追加された商品を縦に並べて表示。

各行の要素：

| 要素 | 説明 |
|------|------|
| 商品名 | 絵文字アイコン＋商品名 |
| 数量コントロール | `−` ボタン・数量表示・`＋` ボタン |
| 小計 | 単価 × 数量 |
| 削除ボタン | `✕` クリックでカートから削除 |

- `−` ボタンで数量が0以下になった場合は自動的にカートから削除
- カートが空の場合「商品を選択してください」メッセージを表示

### 4.3 合計サマリー

| 項目 | 計算式 |
|------|--------|
| 小計 | 全商品の（単価 × 数量）の合計 |
| 消費税 | 小計 × 10%（小数点以下は `Math.round()` で四捨五入） |
| 合計 | 小計 + 消費税 |

---

## 5. 会計ボタン仕様

| ボタン | 動作 |
|--------|------|
| クリア | カートを空にする |
| 現金 | 会計完了アラートを表示（支払方法：現金） |
| カード | 会計完了アラートを表示（支払方法：カード） |
| 会計する | 会計完了アラートを表示（主要アクション、青色） |

会計完了時の処理：
1. アラートダイアログで合計金額と支払方法を表示
2. カートをリセット
3. 伝票番号を+1インクリメント

カートが空の状態で会計ボタンを押した場合は「商品を選択してください」アラートを表示。

## 6. 技術仕様

| 項目 | 内容 |
|------|------|
| 構成 | 単一HTMLファイル（HTML / CSS / JavaScript） |
| スタイリング | カスタムCSS（Tailwindクラスは補助的に使用） |
| 状態管理 | JavaScriptの `cart` 配列（インメモリ） |
| 永続化 | なし（ページリロードでリセット） |
| 外部依存 | なし |
| 対応ブラウザ | モダンブラウザ（Chrome / Firefox / Safari / Edge） |

---

## 7. 主要な状態変数

| 変数名 | 型 | 説明 |
|--------|----|------|
| `allProducts` | Array | 商品マスタ（固定データ） |
| `cart` | Array | カートに入っている商品と数量 |
| `receiptNo` | Number | 現在の伝票番号（初期値: 1） |
